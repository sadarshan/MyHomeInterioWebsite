# MyHOME interi'o' Website

Landing website and studio tool for MyHOME interi'o' (modular kitchen contractor and manufacturer).

## Deploy on Vercel

1. Import this repository in Vercel.
2. Framework preset: `Other`.
3. Build command: leave empty.
4. Output directory: leave empty (root static site).
5. Deploy.

The repository already includes `vercel.json` with:
- security headers
- static asset caching headers
- route rewrites for `/studio` and `/contact`

## SEO Files Included

- `robots.txt`
- `sitemap.xml`
- `site.webmanifest`
- Open Graph/Twitter metadata on key pages
- Structured data (Schema.org JSON-LD)

## Important: Update Final Domain

The current canonical and sitemap URLs use:

`https://www.myhomeinterio.com`

If your final production domain is different, update:

- `index.html`
- `studio.html`
- `contact.html`
- `404.html`
- `robots.txt`
- `sitemap.xml`
